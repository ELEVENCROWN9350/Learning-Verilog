# Learning Verilog

Welcome to my Verilog learning repository!

This repository documents my journey of learning **Verilog HDL**, digital design, and FPGA development. Every project here is written as part of my goal of understanding hardware design from the basics all the way to designing complete processors.

Instead of uploading random code snippets, this repository is organized topic-by-topic as I learn new concepts and implement them in Verilog.

---

## Objectives

- Learn Verilog HDL from scratch
- Understand digital logic design
- Practice writing synthesizable Verilog
- Verify designs using testbenches and simulations
- Build larger digital systems by combining smaller modules
- Progress towards designing a simple CPU
